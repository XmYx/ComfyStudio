Advanced Features

Cinema Shot Designer offers a range of advanced features to enhance your workflow:

Dynamic Parameter Overrides

Use dynamic overrides to automatically update workflow parameters based on previous workflow outputs:

  • Flag parameters to use the previous workflow's image or video outputs.
  • Set parameters to dynamically update once the previous workflow completes rendering.

Multiple Versions

Manage multiple versions of images and videos for each shot:

  • Select different versions from the shots list to view or use specific outputs.
  • Compare versions to track changes or improvements over time.

Extending Clips

Extend existing clips by creating new shots based on previous outputs:

  1. Select a shot and choose Extend Clip(s) from the context menu.
  2. Choose the input parameter to set to the last output of the selected shot.
  3. Create new shots that build upon the existing workflow outputs.

Plugin Support

Enhance Cinema Shot Designer with additional plugins:

  • Install plugins to add new functionalities or integrations.
  • Plugins can be managed and loaded from the plugins directory.

Localization

Use the application in your preferred language:

  • Select the desired language from the Settings dialog.
  • The application interface will update to reflect the chosen language.

Settings Management

Configure application settings to suit your workflow:

  • Set ComfyUI paths, IP addresses, and workflow directories.
  • Save and load workflow parameter defaults for consistency.
