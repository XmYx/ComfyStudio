Getting Started - Cinema Shot Designer

Getting Started

Welcome to Cinema Shot Designer! Follow these steps to create your first shot:

  1. Create a New Project: Go to File > New Project to start a new project.
  2. Add Shots: Click on the Add New Shot button in the toolbar or go to File > Import Shots from TXT to import multiple shots.
  3. Select a Shot: Click on a shot from the list to view and edit its details in the docked panels.
  4. Add Workflows: In the Workflows tab, select and add Image or Video workflows to your shot.
  5. Configure Parameters: Adjust workflow parameters in the Parameters tab to customize the behavior of each workflow.
  6. Render Shots: Use the Render Selected or Render All options from the toolbar or File menu to render your shots.

For detailed instructions on each step, refer to the following sections.

Workflow Management - Cinema Shot Designer

Workflow Management

Workflows are the core processing units in Cinema Shot Designer. They define how your shots are processed and rendered.

Adding Workflows

To add a workflow to a shot:

  1. Select a shot from the shots list.
  2. In the Workflows tab, choose an Image Workflow or Video Workflow from the dropdown.
  3. Click the Add Image Workflow or Add Video Workflow button to attach the workflow to the selected shot.

Configuring Workflows

After adding a workflow, you can configure its parameters:

  • Select the workflow from the list to display its parameters in the Parameters tab.
  • Adjust the parameters as needed to customize the workflow's behavior.
  • Use the Save Workflow Defaults option to save your preferred settings for future use.

Enabling/Disabling Workflows

You can enable or disable workflows:

  • In the Workflows tab, use the checkbox next to each workflow to toggle its enabled state.
  • Disabled workflows will be skipped during the rendering process.

Removing Workflows

To remove a workflow from a shot:

  1. Select the workflow from the list.
  2. Click the Remove Workflow button.
  3. Confirm the removal when prompted.
