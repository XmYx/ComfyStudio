Node Editor with ComfyUI

Cinema Shot Designer leverages ComfyUI as its backend node editor, enabling versatile and complex workflows through node-based configurations.

Understanding Nodes

Each workflow consists of a network of nodes, where each node performs a specific function or transformation on your shot's data.

  • Node Types: Image nodes, Video nodes, Parameter nodes, and more.
  • Inputs and Outputs: Nodes have defined inputs and outputs to pass data through the workflow.

Chaining Workflows

You can chain multiple workflows to process your shots in stages:

  1. Add multiple workflows to a single shot in the Workflows tab.
  2. Each workflow can be configured to take inputs from the outputs of previous workflows.
  3. This allows for complex processing pipelines, such as applying multiple image transformations or video edits sequentially.

Iterating Workflows

Iteration over workflows allows for repetitive processing steps:

  • Looping Nodes: Configure nodes to loop over data for repeated transformations.
  • Dynamic Parameters: Use parameters that change based on previous workflow outputs to create adaptive processing steps.

Creating Custom Nodes

Extend the functionality of ComfyUI by creating custom nodes:

  1. Develop custom node scripts and place them in the plugins directory.
  2. Use the Install/Update Custom Nodes option in the Settings menu to integrate your custom nodes.
  3. Once installed, your custom nodes will be available in the node editor for use in workflows.
