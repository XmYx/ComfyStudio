Frequently Asked Questions (FAQ)

1. How do I add a new shot?

To add a new shot, click on the Add New Shot button in the toolbar or go to File > Import Shots from TXT to import multiple shots from a text file.

2. How can I save my project?

Go to File > Save to save your current project. To save it with a new name or location, choose File > Save As.

3. How do I install custom nodes?

Navigate to Settings > Install/Update Custom Nodes to install or update custom nodes. Ensure that your custom node scripts are placed in the plugins directory before initiating the installation.

4. What should I do if rendering fails?

Check the terminal output for error messages. Ensure that ComfyUI is running and properly configured in the settings. Verify that your workflows are correctly set up and that all required parameters are specified.

5. How can I change the application language?

Open the Settings dialog via Settings > Open Settings, then select your preferred language from the language dropdown menu and click OK.

6. Where are my rendered files saved?

Rendered files are saved in the project directory. You can specify or change the save location in the Settings dialog.

7. How do I update the application?

Visit our website to download the latest version of Cinema Shot Designer. Follow the installation instructions provided there.

8. Can I use external scripts with workflows?

Yes, you can integrate external scripts by adding them as custom nodes in the ComfyUI node editor. Ensure that the scripts are compatible with ComfyUI and placed in the plugins directory.

9. How do I contact support?

If you encounter any issues or have questions, please contact our support team at support@example.com.
