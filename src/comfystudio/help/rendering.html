Rendering

Rendering is the process of generating the final output for your shots using the assigned workflows.

Render Selected Shots

To render specific shots:

  1. Select one or more shots from the shots list.
  2. Click the Render Selected button in the toolbar or choose File > Render Selected.
  3. The selected shots will be added to the render queue and processed sequentially.

Render All Shots

To render all shots in your project:

  1. Click the Render All button in the toolbar or choose File > Render All.
  2. All shots will be added to the render queue and processed one after another.

Stop Rendering

To stop the rendering process:

  1. Click the Stop Rendering button in the toolbar or choose File > Stop Rendering.
  2. The current rendering task will be halted, and the render queue will be cleared.

Render Queue

The render queue manages the list of shots to be processed. You can monitor the progress in the status bar and terminal output.
